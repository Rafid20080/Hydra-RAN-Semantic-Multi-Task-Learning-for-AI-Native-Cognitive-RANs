# Installation Guide for Hydra-RAN

## System Requirements

### Hardware
- NVIDIA GPU with CUDA support (recommended)
- 16GB+ RAM
- 50GB+ storage

### Software
- Ubuntu 20.04+ or Windows 10/11
- Python 3.10+
- CUDA 11.0+ (for GPU acceleration)
- ns-3 with NYU mmWave and NIST 5G/NR modules
